A responsive full-stack web application designed to promote awareness about safe and clean water usage. The project educates users about water purification, health risks of contaminated water, and safe storage practices through interactive and informative content.

Features:
Responsive user-friendly interface, 
Water purification suggestions, 
Educational content on safe water practices, 
Interactive quiz section, 
Backend processing using Flask, 
SQLite database integration for water quality data management.

Technologies Used:
HTML, 
CSS, 
JavaScript, 
Python (Flask), 
SQLite.
